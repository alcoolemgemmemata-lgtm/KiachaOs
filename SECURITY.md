# SECURITY

Responsible disclosure, contact and policies for Kiacha OS.

## Reporting a vulnerability
If you found a security issue, please report it to: security@yourdomain.example (replace with real contact).

Include:
- Affected component
- Steps to reproduce
- PoC (if possible)
- Suggested remediation

## Policy
- We will acknowledge receipt within 3 business days.
- We commit to a coordinated disclosure timeline.
- Do NOT publicly disclose the issue until a fix is available.

## Responsible disclosure process
1. Triage
2. Assign severity
3. Notify maintainers
4. Prepare patch
5. Release and disclose
