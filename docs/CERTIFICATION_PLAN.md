# Certification plan

## Target standards
- FIPS 140-3
- Common Criteria EAL4+
- NIST SP800 series

## Artifacts required
- Design specification
- Security architecture
- Test plan
- Test results
- Source code packages
- Build logs & SBOM
- Vendor statements

## Steps
1. Internal audit
2. Third-party penetration test
3. Submit to lab
4. Address findings
5. Re-test
6. Certification
