# Módulo Militar — Kiacha OS

Este diretório contém a documentação detalhada do módulo MILITAR do Kiacha OS: milestones, épicos, tarefas, arquitetura, SOPs, logs/auditoria, módulos de IA defensiva e protocolos de redundância.

Arquivos principais:
- `milestones.md` — Milestones detalhados (M0..M7)
- `epics.md` — Épicos por milestone
- `tasks_m0.md` .. `tasks_m7.md` — Tarefas e subtarefas executáveis por milestone
- `architecture.md` — Arquitetura recomendada, fluxos e diagrama textual
- `sops.md` — Padrões operacionais e runbooks
- `logs_and_audit.md` — Formato de logs, APIs e evidências forenses
- `ia_defensive.md` — Módulos de IA defensiva e suas APIs
- `fallbacks.md` — Protocolos de fallback, redundância e rollback
- `issues_payload.json` — Payload com issues sugeridas para importação/uso com CLI

Leia `milestones.md` para começar.
